Go Web Framework - Echo

프랑비(Prnagbi)
2021-02-09
Server
1. 서론. 언어에 상관 없이 네트워크 소켓만 열 수 있다면 서버 개발이 가능합니다. 그 중 HTTP를 열어두면 웹 서버가 되는 것이죠. 요즘 언어들은 정형화 된 HTTP 모듈을 기본적으로 제공하기 때문에 간단한 웹 서버 정도는 쉽게 개발 가능합니다. 하지만 기능이 복잡해질수록 프레임워크의 힘이 발휘됩니다. 라우팅, 헤더/파라미터/바디 파싱, JSON 인코드/디코드, 인증, 로깅, 파일 업/다운로드, TLS 등등 많은 것들이 이미 구현되어 제공됩니다. 그것도 오픈소스로. 그럼 Go 웹 프레임워크는 어떠한 것들이 있을까요? https://github.com/mingrammer/go-web-framework-stars 역시나 많습니다.
